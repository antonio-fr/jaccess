  Jaccess
===========

JSBW is python script that readq Jaxx private seed mnemonic in Windows, Linux and MacOS 


## Dependencies

Requires :

* Python 2.7 (not tested on v3)
* pycrypto lib
* qrcode


## Using JSBW

Launch it in a terminal. It will print our your Jaxx private seed.
